กาลครั้งหนึ่งนานมาและ  มีต้นมะม่วง  ต้นมะม่วงต้นนี้มีลักษณะแปลกแตกต่างจากต้นมะม่วงอื่นๆที่เราเคยเห็นกัน
มี ลักษณะ flipchart คือ มีลักษณะ เหลี่ยมๆ แบนๆ ยาวๆ ใหญ่ๆ  และ
มีพ่อค้าเห็นความแปลกของมะม่วง ซึงมีหลายสี เลยไปลิสลงดาเบส เพื่อนำไปเตรียมข้อมูลการขาย
แต่นกพิราบก็บินมาขี้ใส่หัวพ่อค้าเต็มหัวเบย
และขี้นกยังไปตกใส่หัว developer  ที่ยืนอยู่ใกล้ๆ พ่อค้าอีกด้วย และ developer ก็ได้นำผลมะม่วงมาขว้างไล่นกพิราบไปด้วยความโมโห
